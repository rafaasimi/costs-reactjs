# 🎲 Como utilizar

## Clone esse repositório
```sh
git clone https://github.com/rafaasimi/costs-reactjs.git
```


## Navegue até o repositório do projeto
```sh
cd costs-reactjs
```


## Instale as dependências
```sh
yarn install
```

## Inicie o projeto
```sh
yarn start
```

## Inicie o servidor
```sh
yarn server
```
